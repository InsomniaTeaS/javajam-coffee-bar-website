# JavaJam Coffee Bar Website

This is a responsive multi-page coffee shop website I built with HTML and CSS as part of my web-development coursework.

## Pages

- `index.html` - home page
- `menu.html` - coffee menu and pricing table
- `music.html` - monthly music events with audio elements
- `jobs.html` - job application form with required fields
- `javajam.css` - shared responsive styling

## What I practiced

- Semantic HTML structure
- Multi-page navigation
- Responsive layouts with media queries
- CSS Flexbox and Grid
- Tables and forms
- Required form validation
- Audio elements
- Mobile and desktop navigation behavior
